# Utility Interfaces

These interfaces are used as a convinience for various operations.

## `interface IImageCache`

A simple api for accessing images quickly using a cache.

<InterfaceDocumentation interfaceMetadata={IImageCache} />

## `interface ISelection`

Represents the current selection of canvas items.

<InterfaceDocumentation interfaceMetadata={ISelection} />

## `interface Vector2D`

Represents a cartesian (x, y) point on a plane.

<InterfaceDocumentation interfaceMetadata={Vector2D} />

## `interface Type<T>`

Represents a class (similar to the [Type](https://docs.microsoft.com/en-us/dotnet/api/system.type?view=netframework-4.8) class in C#). `CircleBrush` would be `Type<IBrush>`. Essentially [copied from angular](https://github.com/angular/angular/blob/6.1.6/packages/core/src/type.ts#L25).

<InterfaceDocumentation interfaceMetadata={Type} />

## `interface JsonData`

Represents a pure (i.e. serializable) JSON. See [this](https://www.w3schools.com/js/js_json_datatypes.asp) guide for differences between valid javascript object types and JSON types.

<InterfaceDocumentation interfaceMetadata={JsonData} showIndexSignature />
