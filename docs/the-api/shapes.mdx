# Shape Interfaces

These interfaces are used for performing calculations on and rendering geometric shapes.

## `interface IRotatable`

All shapes except circle (which would be redundant) extend this interface which provides a way to easily denote rotation on primitive shapes. Rotation is always measured in degrees and is applied to the center of the element. If `rotation` is omitted then the shape has a rotation of 0 degrees.

<InterfaceDocumentation interfaceMetadata={IRotatable} />

## `interface StyledShape`

All shapes (except Line) extend this interface which provides optional styling information to a shape. When rendering, the [painter api](/painterapi) uses these attributes to determine styling information. All attributes are optional and will be filled with default values if null.
