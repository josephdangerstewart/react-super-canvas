# The `Renderable` interface

`Renderable` is a single serializable representation for any [canvas item](/icanvasitem) on the canvas. When `onChange` is called, all canvas items are converted to renderables. It encodes any rendered output from the [painter api](/painterapi) along with any state data returned by a canvas item's `toJson` method. `Renderable` is also useful because it provides a fallback for when the super canvas can't map serialized data back to it's original canvas item (for instance, if you are loading data into a super canvas but the super canvas doesn't have all of the canvas item classes for that data). It should be noted that outputed data from the super canvas in the form of the renderable interface should be treated as **read only**.
