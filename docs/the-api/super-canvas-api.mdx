# Super Canvas Manager

This document lays out the API that manages the virtual canvas space, mediates user interactions, and renders canvas items. It is not meant to be extended directly so it might not make it in the final version of documentation. Right now these are just development notes

## Requirements

### Required methods

### Required state

1. `activeCanvasItems: CanvasItem[]` - The canvas items that are in the editor
2. `painter: IPainterAPI` - The api for drawing to the canvas
3. `panOffset: Vector2D` - The offset factor due to panning
4. `scale: Number` - The scale factor due to scrolling
